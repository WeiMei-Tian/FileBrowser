# FileBrowser
##简易文件浏览器app<br>

数据加载采用loader机制，数据加载与界面展示分离，通过loader回调交互<br><br>

列表展示使用drakeet的开源项目MultiType<br>
其他库引用：<br>
ButterKnife <br>
Logger <br>
EventBus <br>
